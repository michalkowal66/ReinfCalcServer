Reinforcement Calculator Project

About the project

Reinforcement Calculator is a desktop application that can be used for dimensioning simple reinforced concrete elements. The app is written in Python using PyQt5, requests, json, and math modules. It communicates with a Django server, that comes with a calculation engine that produces the task results. The server back-end part is written in Python, the front-end part uses Django built-in templating system, Bootstrap4, and MathJax.

All the sources (Desktop App, Django Server, Calculation Engine) are available here.

Using the application

After installing all required modules listed in requirements.txt file and downloading the current code, the application can be started by running main.py script

  • Choosing the element to work with

    • On start of the application the user is welcomed with the log in screen. With the information used to create the profile user can log in to the application (the server must be online at that time to allow authentication of the user).

    Once logged in, a main window of the application shows up letting user choose from 4 types of reinforced concrete elements:

    • Plate
    • Beam
    • Column
    • Foundation foot

    In order to begin working with a certain element its corresponding button must be clicked.

  • Providing necessary data

    • Next step is to provide all necessary information about the element, which differ depending on the chosen element type. All fields must be filled bearing in mind units stated in the corresponding labels.

  • Using built-in resources

    • For users' convenience a set of tables containing useful technical parameters has been included in the application. It can be accessed either by selecting Resources->Included Data from the menu bar, or using ctrl + i keyboard shortcut.

  • Running calculations

    • All the applications are done server-side, in order to obtain the results for the task user has to ensure that following conditions are fulfilled:

    • The server is online
    • The user has stable internet connection

    If so, in order to obtain the simplified results user has to press a button.

  • Generating a report

    • Users can access the details of a last calculated task via the application pressing a button.

    Doing so will open the default web browser and load a report page for the last calculated task.

    If the user is not logged in the log in page will be displayed first and redirect the user after successful authentication.

  • Saving and loading results

    • At any time of working with the element, users can save their progress using save option by selecting File->Save from the menu bar, or using ctrl + s shortcut. The application uses JavaScript Object Notation to save the information about the user's progress.

Using the website

After deploying the Django server using hosting services, or running it locally on the host device users gain access to the full functionality of the desktop application. The server comes with an account management system (users registration and login), a calculation engine, a task results handling system, and an admin panel.

  • Site navigation

    • To navigate on the site user should use links in a navbar and a footer.

  • Creating an account

    • In order to create an account user should visit the SignUp page using an appropriate link in the navbar.

    After successfully creating an account, the user is taken to the log in page.

  • Accessing the results

    • To access the results user first needs to log in.

    Once the user has logged in, the navbar displays additional link - Results. After the user does any calculations they'll show up there.

    To access detailed results of calculations, the user has to select task and open the report page using the button.
