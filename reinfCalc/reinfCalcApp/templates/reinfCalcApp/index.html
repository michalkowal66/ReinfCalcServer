Welcome to ReinfCalc!

Reinforcement Calculator is a desktop application that can be used for dimensioning simple reinforced concrete elements.

To start visit the About page.

1. Create an account

You can create an account using the options in the navbar.

2. Get sources

Download the latest version of the application, and install the requirements according to the instructions.

3. Choose an element to work with

Log in to the application with your account. Choose one of the available types of elements.

4. Calculate it and check results

After providing all the necessary data run the calculations. You'll be able to access the results both in the application (simplified) and the website (detailed report)!
